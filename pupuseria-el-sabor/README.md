# Pupusería El Sabor - Landing Page

Landing page interactiva para una pupusería salvadoreña, desarrollada como proyecto del curso de Transformación Digital para la Docencia Técnica.

## 🚀 Características

- Diseño responsive (móvil, tablet, desktop)
- Menú de navegación fijo con menú hamburguesa
- Formulario de reservas con validación en JavaScript
- Animaciones al hacer scroll
- Secciones: Inicio, Nosotros, Menú, Reservas
